# Beginners-JS

A collection of small JavaScript projects for beginners. Each project is designed to help you learn and practice JavaScript, HTML, and CSS in a fun and interactive way.

## Projects Included

| Project Name           | Description |
|------------------------|-------------|
| Color Flipper         | Changes the background color randomly. |
| Counter App           | A simple counter to track numbers. |
| Digital Clock         | Displays a real-time digital clock. |
| Random Quote Generator | Displays random quotes with each click. |
| Scissors-Paper-Rock   | A simple game against the computer. |
| Tic Tac Toe           | A classic Tic Tac Toe game. |
| Tip Counter           | A simple app to calculate tips. |

## How to Use

1. Clone the repository:
   ```sh
   git clone https://github.com/Abhimannu09dev/beginners-JS.git
   ```
2. Navigate into the folder:
   ```sh
   cd beginners-JS
   ```
3. Open any project folder and run the `index.html` file in your browser.
